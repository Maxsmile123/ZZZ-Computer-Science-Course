# Реферат по курсу "Фундаментальная информатика"

Реферат представляет собой углублённое задание по темам, изученным в курсе.
В рамках реферата необходимо самостоятельно изучить тему.

---

## Варианты
*Вы в праве предложить свой вариант. Важно, чтобы тема относилась к программе семестра. Ниже примеры заданий, которые можно взять*.
1) Реализация Brute Force подбора пароля при помощи Hudra
2) Изучение фреймворка Burp Suite 
3) Реализовать машину Тьюринга на языке Си
4) Реализовать программу перевода МТ с чётвёрок в пятёрки и обратно
5) Сравнение скорости компиляции С++ кода на различных компиляторах
6) Написать программу преобразования из недетерминированного автомата в детерминированный
7) Реализация алгоритма быстрого перемножения матриц
8) Динамический анализ программ при ревёрсе программ
9) Автоматический анализ программ при ревёрсе программ
10) Бинарный патчинг в ревёрсе программ
11) Решение 8 CTF задач на Assembler
12) Реверс байт кода Java, .NET, Python
13) Реализация TCP-чата с использованием C++ Boost.Beast/Boost.Asio
14) Реализация защиты от Arp-spoofing
15) Реализация класса Матрица на С++ с использованием умных указателей
